FTC Team Website Training

For Middle School Robotics Students

Goal: Build a simple, clean website for your FTC team using HTML and a little CSS.

1. What is a website?

A website is a group of pages you can open in a browser (like Chrome or Edge). Each page is usually written in a language called HTML.

Your FTC team website can show:

  • Team name, number, and logo
  • Pictures of your robot and events
  • Meet the team members
  • Engineering notebook highlights
  • Contact info and social media

2. Basic HTML structure

Every HTML page has a basic “skeleton” like this:

<!DOCTYPE html>
<html>
  <head>
    <title>My FTC Team</title>
  </head>
  <body>
    <h1>Welcome to Our FTC Team Website!</h1>
    <p>We are Team 12345.</p>
  </body>
</html>

Try this:

  • Create a file called index.html
  • Copy the skeleton above into it
  • Open it in a browser and see your first web page

3. Important HTML tags

  • <h1>...</h1> – Big heading (page title)
  • <h2>...</h2> – Section heading
  • <p>...</p> – Paragraph of text
  • <ul> and <li> – Bulleted list
  • <img> – Image (picture)
  • <a> – Link to another page or site
  • <div> – Box/container for grouping content
  • <footer> – Bottom section of the page
<h1>FTC Team 12345</h1>
<p>We love building robots and learning new things!</p>

<h2>Our Achievements</h2>
<ul>
  <li>Inspire Award Finalist</li>
  <li>Winning Alliance Captain</li>
</ul>

4. Adding images and links

4.1 Images

Save your image in the same folder as your HTML file, for example robot.jpg.

<img src="robot.jpg" alt="Our FTC Robot" width="300">

4.2 Links

<a href="https://www.firstinspires.org/robotics/ftc" target="_blank">
  Learn more about FTC
</a>

Try this:

  • Add a picture of your robot to the page
  • Add a link to your team’s YouTube or Instagram

5. Adding simple CSS for style

CSS controls how your website looks (colors, fonts, spacing). Here is a simple style block you can put inside your <head>. 

<style>
  body {
    font-family: Arial, sans-serif;
    background: #f5f7fb;
    color: #222;
  }
  header {
    text-align: center;
    padding: 1rem 0;
  }
  h1, h2 {
    color: #1b4b9c;
  }
</style>

Try this:

  • Change the background color
  • Change the heading color to your team color

6. Starter layout for an FTC team website

Here is a simple template you can customize for your team.

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>FTC Team 12345</title>
  <style>
    body { font-family: Arial, sans-serif; background:#f5f7fb; margin:0; }
    header, footer { background:#1b4b9c; color:white; text-align:center; padding:1rem; }
    .content { padding:1rem 1.5rem; }
    .section { background:white; margin-bottom:1rem; padding:1rem; border-radius:8px; }
  </style>
</head>
<body>
  <header>
    <h1>FTC Team 12345</h1>
    <p>Team Name – City, State</p>
  </header>

  <div class="content">
    <div class="section">
      <h2>About Our Team</h2>
      <p>We are a middle school FTC team that loves robots and learning.</p>
    </div>

    <div class="section">
      <h2>Our Robot</h2>
      <img src="robot.jpg" alt="Our Robot" width="300">
      <p>This is our robot for the current season.</p>
    </div>

    <div class="section">
      <h2>Meet the Team</h2>
      <ul>
        <li>Student 1 – Role</li>
        <li>Student 2 – Role</li>
        <li>Student 3 – Role</li>
      </ul>
    </div>

    <div class="section">
      <h2>Contact Us</h2>
      <p>Email: team12345@example.com</p>
      <p>
        <a href="https://www.youtube.com" target="_blank">YouTube</a> |
        <a href="https://www.instagram.com" target="_blank">Instagram</a>
      </p>
    </div>
  </div>

  <footer>
    <p>© 2026 FTC Team 12345</p>
  </footer>
</body>
</html>

7. Team practice checklist

  • Create an index.html file with your team name and number
  • Add an “About Our Team” section
  • Add at least one robot picture
  • Add a “Meet the Team” list with names and roles
  • Add links to your social media or videos
  • Customize colors to match your team branding

Next steps

Once your basic site works, you can add more pages: Robot Design, Outreach, Season Highlights, and more.
